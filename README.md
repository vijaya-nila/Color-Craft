**ColorCraft**
A modern color picker tool built using HTML, CSS, and JavaScript. 
Users can adjust Hue, Saturation, and Lightness sliders to generate colors in real-time.

**Key Features**
-  Real-time color preview engine  
-  Multi color modes (RGB, HSL, CMYK)  
-  Image-based color extraction  
-  Advanced eyedropper picker  
-  Smart palette generator  
-  Accessibility contrast checker  
-  Save and manage colors  
-  Light/Dark UI toggle  
-  Fun interactive mode

  **Tech Stack**
- HTML  
- CSS  
- JavaScript
